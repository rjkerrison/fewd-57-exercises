  • arrays

    Lists, collections. If you want to get a value, use square brackets and an index number. Remember: indexes begin at 0.

    const greetings = [
  'hello',
  'how are you?',
  'how you doing?',
  'g'day mate'
]

greetings[2] // 'how you doing?'
greetings[0] // 'hello'

  • objects

    For structured data. Get values out (or set them to new values) using dot notation or square bracket indexes.

    let simple = {a: 5, b: 6}

simple.b = simple.a + 10
simple.b // 15

const meteo = {
  location: 'London',
  currentWeather: 'rain'
}

meteo.location // 'London'
meteo['currentWeather'] // 'rain'

  • more objects

    Objects can get complicated.

    const personalDetails = {
  'name': 'Robin',
  'cats': [
    {
      name: 'Mr Snuggles',
      age: 6
    },
    {
      name: 'Captain Earhart',
      age: 5
    }
  ]
}

personalDetails.cats[1].name // 'Captain Earhart'
personalDetails.cats[1].name = 'Lieutenant Jemison'
personalDetails.cats[1].name // 'Lieutenant Jemison'

  • class

    Custom datatypes for storing data and creating methods (custom functions)

    class Person {
  constructor(name, age) {
    this.name = name
    this.age = age
  }
  getAge() {
    return this.age
  }
  celebrateBirthday() {
    this.age += 1
  }
}

const robin = new Person('Robin', 15)
robin.getAge() // 16
robin.celebrateBirthday()
robin.getAge() // 16

  • not quite existing

    Before something exists, it is undefined. If we want to blank something out, we can set it to null.

    null
undefined

Variables

  • Declaration

    When declaring a variable, const or let must be used.

    const name = 'Robin'
let age = 26

  • Modifying

    Any let variable can be overriden so that it can be updated to a new value

    const age = 26
// after a birthday
age = 27

  • Using them

    To use a variable, write its name.

    const occupation = 'Muffin Man'
const streetAddress = 'Drury Lane'
const question = `Do you know the ${occupation} who lives on ${streetAddress}?`
console.log('streetAddress') // 'streetAddress'
console.log(streetAddress) // 'Drury Lane'
console.log(question)

Number operators

  • Addition

    As you might expect, this is done by using + between two number values.

    const apples = 3
const oranges = 5
const target = document.getElementById('orange-apple-sum')
target.innerText = apples + oranges

  • Division

    Perhaps still your first guess: this is done by using / between two number values.

    const winOccurences = 97
const totalAttempts = 300
const target = document.getElementById('leroy-odds')
target.innerText = winOccurences / totalAttempts

  • Multiplication

    Possibly logical, we use * between two number values.

    const nintendoPrice = 110
const numberOfDsConsoles = 6
const target = document.getElementById('total-price')
target.innerText = nintendoPrice * numberOfDsConsoles

Strings

  • Concatenation

    This might be a suprise, but + between two string values concatenates them.

    const firstFruit = 'oranges'
const secondFruit = 'lemons'
const target = document.getElementById('st-clements')
target.innerText = firstFruit + secondFruit

  • Template strings

    A recent innovation in the JavaScript spec, template strings (also called string interpolation) allow us to create strings with variables in them.

    const currentClass = 'FEWD 57'
const message = `Hello, ${currentClass}!`
const target = document.getElementById('hello')
target.innerText = message
