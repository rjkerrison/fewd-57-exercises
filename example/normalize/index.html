My Webpage, Without Styling

I've created this webpage to demonstrate what complicated HTML looks like without styling.

Differences Section

There are some major differences between the user agent stylesheets of different browsers, such as

  • How much margin there is on the body
  • How large the font of an h1 is inside a section
  • Borders and text-decoration of an abbr element
  • The size of small text
  • How sup and sub elements affect the line-height
  • The padding inside a on Chrome on macOS

User-defined differences

As well as the differences between browsers, users can set their preference for the default font-size and default font-family of webpages.

We can always override these ourselves with our own CSS.
